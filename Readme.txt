Getting Started
1. Download the required dataset here (http://snap.stanford.edu/data/amazon/productGraph/categoryFiles/reviews_Cell_Phones_and_Accessories_5.json.gz).
2. For each script, ensure the correct path to the data.
3. Create a virtual environment using the following command: python3 -m venv venv
4. Activate virtual environment with this command: source venv/bin/activate
5. Install the requirements with this command: pip install -r requirements.txt

Project Directory
The scripts are categorized in folders.
After the virtual environment have been created, each script can be run independently.
Before running each script, ensure it has the correct file path to the json data downloaded.
To run any script, the command is as follows:
  python <path to script>
